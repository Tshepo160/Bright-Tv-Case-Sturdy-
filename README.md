# 📺 Bright TV Case Study

## 🚀 Overview

The **Bright TV Case Study** is an end-to-end data analytics project designed to simulate a real-world media analytics workflow. It covers the full pipeline from data preparation and transformation to visualization and business insights.

This project demonstrates strong skills in:

* SQL data analysis
* Data cleaning & transformation
* Dashboard creation
* Business storytelling

---

## 🎯 Objectives

* Analyze TV viewership data
* Identify user behavior patterns
* Discover top-performing channels and content
* Build a clean, interactive dashboard for stakeholders

---

## 🧩 Project Workflow

### 🔴 1. Planning & Architecture

* Designed data flow using diagrams
* Identified data sources
* Structured SQL environment
* Prepared raw data for ingestion

**Key Tasks:**

* Miro Diagram (Data Flow Design)
* Data Source Identification
* Convert Excel → CSV
* Data Ingestion Setup

---

### 🔵 2. Visualization

* Created insights using charts and dashboards
* Built stakeholder-ready presentation

**Key Tasks:**

* Pivot Charts & Graphs (Excel)
* Final Presentation (Canva)

---

## 🗂️ Dataset

The dataset includes:

* User IDs
* Viewing channels
* Watch duration
* Demographics (age, gender, region, ethnicity)
* Dates and timestamps

---

## 📊 Key Analysis Performed

* Total viewership trends
* Regional growth analysis
* Gender distribution by age group
* Ethnicity vs gender insights
* Top 10 channels by audience
* Viewing duration per channel
* Peak viewing times

---

## 🛠️ Tools & Technologies

* **SQL** – Data querying & transformation
* **Excel** – Pivot tables & charts
* **Canva** – Presentation design
* **Data Visualization Techniques** – Dashboard storytelling

---

## 📈 Dashboard Features

* Interactive filtering (region, gender, age)
* KPI summaries
* Trend analysis visuals
* Top-performing content insights

---

## 💡 Key Insights

* Certain channels dominate specific demographics
* Viewing patterns vary significantly by region
* Peak engagement occurs at specific times of day
* Audience segmentation is critical for content strategy
